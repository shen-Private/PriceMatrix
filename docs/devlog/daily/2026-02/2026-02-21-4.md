## 學到的概念

**CSS 管理三種方式**
- Inline Style（JS 物件）：作用域隔離，但無法用 `:hover`、media query
- 一般 CSS（.css）：全域生效，class 名稱可能撞名
- CSS Modules（.module.css）：局部生效，打包時自動加 hash，業界主流

**`.module.css` 的點不是裝飾**
打包工具靠 `.module.css` 這個關鍵字來決定是否啟動 CSS Modules 處理邏輯，不能換成 `-` 或 `_`

**CSS 分層管理**
- `src/styles/global.css`：body、reset、CSS 變數（全域）
- `src/modules/xxx/xxx.module.css`：元件局部樣式
- Header/Footer 雖然視覺上是全域，CSS 還是局部的，跟著元件走

**檔名的 `.` 邏輯**
`主名稱.性質.副檔名`，例如 `DiscountPanel.module.css`、`App.test.js`，從右到左讀：這是CSS → 這是模組 → 元件名稱
