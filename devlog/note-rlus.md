
## 💡 建議的記錄習慣

### 每天結束前（5 分鐘）
```
開啟：devlog/daily/2026-02/20260201.md

寫下：
1. 今天做了什麼（流水帳）
2. 遇到什麼問題
3. 明天要做什麼

然後：
git add .
git commit -m "日誌: 2026-02-01"
git push
```

### 學到新東西時（隨時）
```
開啟：devlog/learning/對應的主題.md

補充：
- 新學到的概念
- 實用的指令
- 踩過的坑

隨時更新，不用每天
```

### 做重要決定時（偶爾）
```
建立：devlog/decisions/新檔案.md

記錄：
- 要解決什麼問題
- 考慮了哪些方案
- 為什麼選這個
- 預期的優缺點

未來可以回顧
```
