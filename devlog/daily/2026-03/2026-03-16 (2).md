### sales 模組報價單實作

#### DB
```sql
CREATE TABLE sales_quote (
    id BIGSERIAL PRIMARY KEY,
    customer_id BIGINT NOT NULL REFERENCES pricing_customers(id),
    status VARCHAR(20) NOT NULL DEFAULT 'DRAFT',
    note VARCHAR(500),
    created_by VARCHAR(100) NOT NULL,
    created_at TIMESTAMP NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMP NOT NULL DEFAULT NOW(),
    parent_quote_id BIGINT REFERENCES sales_quote(id)
);

CREATE TABLE sales_quote_item (
    id BIGSERIAL PRIMARY KEY,
    quote_id BIGINT NOT NULL REFERENCES sales_quote(id),
    product_id BIGINT NOT NULL REFERENCES pricing_products(id),
    quantity INT NOT NULL,
    unit_price DECIMAL(10,2) NOT NULL
);
```

## 踩到的坑

### `process.env` 在 Vite 不能用
**症狀：** `Uncaught ReferenceError: process is not defined`

**原因：** CRA 用 `process.env`，Vite 用 `import.meta.env`。

**解法：** 改為 `import.meta.env.VITE_API_URL ?? ''`
