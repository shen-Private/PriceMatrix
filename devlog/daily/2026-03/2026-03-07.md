## 踩到的坑

### CORS 缺少 PATCH method
**症狀：** 停用帳號時前端報 CORS 錯誤。

**原因：** `CorsConfig.java` 的 `allowedMethods` 沒有列 `PATCH`。

**解法：** 加上 `"PATCH"`。

### created_at 500 錯誤
**症狀：** 新增帳號 API 回傳 500，錯誤 `null value in column "created_at"`。

**原因：** `CommonUser.java` 的時間欄位沒有自動填值。

**解法：** 加上 `@CreationTimestamp` / `@UpdateTimestamp`。

### Lombok boolean 命名陷阱
**症狀：** `isActive` 欄位讓 Lombok 產生 `isIsActive()` getter，Service 無法編譯。

**原因：** `boolean` 欄位用 `is` 開頭，Lombok 會再加一層 `is`。

**解法：** DTO 欄位改名為 `active`，getter 就會是 `isActive()`。

### 環境變數名稱不一致
**症狀：** 線上環境前端一直連 localhost:8080。

**原因：** `authApi.ts` 用 `REACT_APP_API_BASE_URL`，但 `.env.production` 裡是 `REACT_APP_API_URL`。

**解法：** 統一改成 `REACT_APP_API_URL`。
