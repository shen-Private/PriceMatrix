## 踩到的坑

### axios withCredentials 未設定
**症狀：** API 回傳 CORS 401，Cookie 沒有帶過去。

**原因：** axios 跨域請求預設不帶 Cookie。

**解法：** `axios.defaults.withCredentials = true;`

### JwtAuthFilter 擋住 CORS preflight
**症狀：** `POST /api/inventory/transactions/adjust` 被 CORS 擋，前端看到 No Access-Control-Allow-Origin。

**原因：** OPTIONS preflight 不帶 Cookie，JWT Filter 判定 token = null 回傳 401，導致 CORS 回應頭不存在。

**解法：**
```java
if ("OPTIONS".equalsIgnoreCase(request.getMethod())) {
    filterChain.doFilter(request, response);
    return;
}
```

### inventory_stock 重複資料
**症狀：** `getOverview()` 回傳 500，`NonUniqueResultException: 4 results were returned`。

**原因：** seed SQL 執行多次，`item_id` 沒有 UNIQUE 制約，重複插入不報錯。

**解法：**
```sql
ALTER TABLE inventory_stock ADD CONSTRAINT uq_inventory_stock_item_id UNIQUE (item_id);
```

### seed SQL AUTO INCREMENT ID 不固定
**症狀：** `inventory_stock` INSERT 出現 FK 錯誤，`item_id=1` 不存在。

**原因：** DELETE 後重新 INSERT，`inventory_item` 的 id 從 66 開始而非 1。

**解法：** seed 改用 `SELECT id FROM inventory_item WHERE product_id = ?` 動態取得 id。

### psql 文字化け（SJIS / UTF-8 衝突）
**症狀：** 日文錯誤訊息亂碼。

**原因：** Windows 的 psql 客戶端使用系統 locale（SJIS），DB 是 UTF-8。

**解法：**
```
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d pricematrix --set=client_encoding=UTF8
```
