## 踩到的坑

### `|| ''` vs `?? ''`
**症狀：** `.env.production` 設為空字串，但 Vercel build 後仍打 localhost。

**原因：** `||` 把空字串視為 falsy，fallback 到 `'http://localhost:8080'`。

**解法：** 改用 `??`（Nullish Coalescing），只有 `null` / `undefined` 才 fallback。

### Safari 跨域 Cookie
**症狀：** Chrome 電腦版正常，iPhone Safari / Chrome 無法登入後呼叫 API。

**原因：** Safari 對第三方 Cookie 限制更嚴，`SameSite=None` 也可能被擋。

**解法：** Vercel Proxy 讓前後端同域，Cookie 變成第一方。

### pending 商品沒有 inventory_item
**症狀：** 簡易建檔後再掃同一條碼，找不到商品（404）。

**原因：** `handleQuickAdd` 只呼叫 `POST /api/products/pending`，barcode 沒有存進 `inventory_items`。

**解法：** 簡易建檔後第二步呼叫 `POST /api/inventory/items/pending`，同步建立 item + stock。

### InventoryItem.unit NOT NULL
**症狀：** `POST /api/inventory/items/pending` 回 500，`not-null property references a null: unit`。

**原因：** `createPendingItem` 沒有設定 `unit`。

**解法：** `item.setUnit("個")` 設定預設值。

### `<tr>` 內空白文字節點
**症狀：** React warning `whitespace text nodes cannot be a child of <tr>`。

**原因：** `<React.Fragment>` 與 `<tr>` 之間有空行；`</th>` 後面有空格再接註解。

**解法：** 移除空行與多餘空格。
