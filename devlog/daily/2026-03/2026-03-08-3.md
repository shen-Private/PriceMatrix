## 踩到的坑

### psql 文字化け（Windows 11 SJIS）
**症狀：** categories 查詢結果亂碼，部分資料出現 SJIS 轉換錯誤。

**原因：** Windows 11 日文環境系統 locale 預設 SJIS，psql client 跟著走。

**解法：** 控制台 → 地區 → 系統管理 → 變更系統地區設定 → 勾選「Beta：使用 Unicode UTF-8 提供全球語言支援」→ 重新開機。改完後 psql 不需要再加 `--set=client_encoding=UTF8`。

### manufacturers 重複插入
**症狀：** INSERT 執行兩次，Canon / OKI / Epson 各出現兩筆。

**解法：** `DELETE FROM manufacturers WHERE id IN (4, 5, 7);`
