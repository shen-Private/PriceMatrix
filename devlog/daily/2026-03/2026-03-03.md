## 踩到的坑

### FetchType.LAZY → 500 錯誤
**症狀：** GET transaction API 回傳 500，錯誤訊息 `ByteBuddyInterceptor`。

**原因：** `InventoryTransaction` 的 `item` 欄位是 `FetchType.LAZY`，Jackson 序列化時無法存取 Hibernate proxy。

**解法：** 改成 `FetchType.EAGER`。

### Render 沒有 auto-deploy
後端 push 後 Render 沒有自動觸發，需手動 deploy。注意確認 deploy 的 commit 是否為最新版本。
